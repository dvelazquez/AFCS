# AFCS
<h3>Auto Function Checksheet System</h3>  
Intended to run functional checks to IPCs with Image Processing, OpenCV and [TBD Hardware]  

Compiling with CMakeLists instead of Makefile:  
	cd build  
	cmake ..  
	make -j 4  
	./AFCS.bin   (or make -j 4 && ./AFCS.bin
